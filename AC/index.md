# 电子电路基础 学习笔记

> 模拟电路绝对是电路学习中浓墨重彩的一笔. 它向人们介绍了不同的非线性元件, 而工程师们正是利用非线性元件才创造出了生活中各种各样的电路. 它的内容相当丰富, 这也造就了学习中的困难重重.

本笔记基于教材与教学顺序, 分为线性/非线性电子电路两个部分. 其中, 在非线性部分中, 只介绍教材中的一部分内容. 所谓 "线性" 与 "非线性", 就是人们利用非线性元件时, 输入的信号类型不同. 而不同的信号类型, 也对应着不同模型的使用.

本笔记的参考教材为 冯军, 谢嘉奎, 王蓉等. 电子线路 线性部分. 6版. 北京: 高等教育出版社, 2022. 与 冯军, 谢嘉奎, 王蓉等. 电子线路 非线性部分. 6版. 北京: 高等教育出版社, 2021. 很经典的一套教材.
